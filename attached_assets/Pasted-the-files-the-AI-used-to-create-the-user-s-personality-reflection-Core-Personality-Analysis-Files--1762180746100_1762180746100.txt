Core Personality Analysis Files:

2. server/memory/MemoryManager.ts - Memory orchestration
   * processMessage() - Processes all user messages for memory extraction
   * getComprehensiveContext() - Retrieves full user context
3. server/memory/SemanticMemoryService.ts - Memory extraction
   * extractAndStoreMemories() - Extracts personality facts from conversations
   * analyzeContentForMemories() - AI analysis of content
4. server/memory/MemoryRetrievalService.ts - Memory retrieval
   * getContextualMemories() - Gets relevant memories for context
   * getConversationRelevantMemories() - Retrieves conversation-specific memories
5. server/memory/ConversationContinuityService.ts - Session management
   * Maintains context across sessions
   * Tracks conversation threads and topics
6. server/memory/MemoryAnalyticsService.ts - Generates insights
   * generateMemoryInsights() - Creates therapeutic insights from memories

Data Sources Used:
The AI pulls from these data points:
* Journal entries (storage.getJournalEntries())
* Chat messages (storage.getUserMessages())
* Mood entries (storage.getMoodEntries())
* Semantic memories (storage.getUserMemories())
* User facts (storage.getUserFacts())
* Memory insights (storage.getMemoryInsights())
